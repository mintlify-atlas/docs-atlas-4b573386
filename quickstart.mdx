## Create your account

Follow these steps to join your college community on Echo.

<Steps>
  <Step title="Navigate to signup">
    Go to the Echo homepage and click **Sign Up** or navigate directly to `/auth/signup`.
  </Step>

  <Step title="Choose a username">
    Pick a unique username that will be your identity on Echo.

    <Info>
      Usernames must be 5-50 characters long and can contain letters, numbers, dots, and underscores.
    </Info>

    The system will check availability in real-time as you type.
  </Step>

  <Step title="Enter your college email">
    Provide your official college email address. Echo will validate that it belongs to a supported institution.

    ```typescript
    // Email validation example
    const email = "student@college.edu";

    // Echo validates the email domain
    const response = await fetch(`/api/validateCollege?email=${email}`);
    const { name, id } = await response.json();
    // Returns: { name: "College Name", id: "college_id" }
    ```

    <Warning>
      Only verified college email domains are accepted. Contact your administrator to add your college.
    </Warning>
  </Step>

  <Step title="Verify your email">
    Click **Send Code** to receive a 15-character verification code via email.

    Check your inbox and paste the code into the verification field, then click **Verify**.

    ```typescript
    // Verification flow
    // 1. Request verification code
    await fetch('/api/verifyCollege/send', {
      method: 'POST',
      body: JSON.stringify({ email })
    });

    // 2. Verify the code
    await fetch('/api/verifyCollege/verify', {
      method: 'POST',
      body: JSON.stringify({ email, code: '15-char-code' })
    });
    ```
  </Step>

  <Step title="Set your password">
    Create a strong password that meets these requirements:

    - 8-50 characters long
    - At least one uppercase letter
    - At least one lowercase letter
    - At least one number
    - At least one special character

    <Tip>
      Your password is never stored in plain text. Echo uses cryptographic hashing to protect your credentials.
    </Tip>
  </Step>

  <Step title="Complete signup">
    Click **Sign up** to create your account. You'll be automatically logged in and redirected to your feed.
  </Step>
</Steps>

## Navigate the interface

Once logged in, you'll see the main dashboard with several key areas:

<Tabs>
  <Tab title="Feed">
    The main feed displays posts from your college community. Posts are shown in reverse chronological order.

    - **Public posts**: Visible to anyone in your college
    - **Followers-only**: Only visible to users you follow
    - **Mentioned**: Only visible to mentioned users
  </Tab>

  <Tab title="Profile">
    Access your profile by clicking your username. Here you can:

    - View your posts and replies
    - See your reposts
    - Edit your bio and privacy settings
    - Check follower/following counts
  </Tab>

  <Tab title="Notifications">
    Get notified when:

    - Someone likes your post
    - Someone replies to your post
    - Someone follows you
    - Someone reposts your content
    - Admins make announcements
  </Tab>

  <Tab title="Search">
    Search for:

    - Other users by username
    - Posts by content
    - Specific colleges
  </Tab>
</Tabs>

## Create your first post

<Steps>
  <Step title="Open the composer">
    Click the **New Post** button or the compose icon in the navigation.
  </Step>

  <Step title="Write your content">
    Type your message. You can include:

    - Text content (up to the character limit)
    - Media attachments (images)
    - Mentions using `@username`

    <CodeGroup>
    ```typescript API Call
    // Create a post via API
    const response = await fetch('/api/post', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        content: 'Hello Echo!',
        media: ['https://blob.url/image.jpg'],
        privacy: 'ANYONE'
      })
    });

    const post = await response.json();
    ```

    ```typescript Privacy Options
    type PostPrivacy = 
      | 'ANYONE'      // Visible to all college users
      | 'FOLLOWED'   // Only followers can see
      | 'MENTIONED'; // Only mentioned users can see
    ```
    </CodeGroup>
  </Step>

  <Step title="Add media (optional)">
    Click the image icon to upload photos. Echo automatically:

    - Scans images for NSFW content using AI
    - Optimizes and stores media in Vercel Blob
    - Generates public URLs for your attachments

    <Warning>
      Images that fail NSFW detection will be rejected. This helps keep Echo safe for everyone.
    </Warning>
  </Step>

  <Step title="Set privacy level">
    Choose who can see your post:

    - **Anyone**: All users in your college
    - **Followers only**: Only people following you
    - **Mentioned only**: Only users you mention
  </Step>

  <Step title="Publish">
    Click **Post** to publish. Your post will appear in your feed and your followers' feeds.

    <Tip>
      Rate limits apply: 27 posts per minute. This prevents spam and abuse.
    </Tip>
  </Step>
</Steps>

## Interact with posts

Echo provides several ways to engage with content:

<Accordion title="Like and dislike">
  Show appreciation or disagreement with the reaction buttons:

  ```typescript
  // Toggle like on a post
  await fetch('/api/post/toggle/like', {
    method: 'POST',
    body: JSON.stringify({ postId: 'post_id' })
  });

  // Toggle dislike on a post
  await fetch('/api/post/toggle/dislike', {
    method: 'POST',
    body: JSON.stringify({ postId: 'post_id' })
  });
  ```

  Likes and dislikes are mutually exclusive - you can't do both on the same post.
</Accordion>

<Accordion title="Reply to posts">
  Start conversations by replying:

  ```typescript
  // Reply to a post
  await fetch('/api/post/reply', {
    method: 'POST',
    body: JSON.stringify({
      parentPostId: 'original_post_id',
      content: 'My reply',
      privacy: 'ANYONE'
    })
  });
  ```

  Replies create threaded conversations that users can explore.
</Accordion>

<Accordion title="Repost content">
  Share posts with your followers:

  ```typescript
  // Repost to your timeline
  await fetch('/api/post/toggle/repost', {
    method: 'POST',
    body: JSON.stringify({ postId: 'post_id' })
  });
  ```

  Reposts appear in your feed and notify the original author.
</Accordion>

<Accordion title="Follow users">
  Follow interesting users to see their posts in your feed:

  ```typescript
  // Follow/unfollow a user
  await fetch('/api/follow', {
    method: 'POST',
    body: JSON.stringify({ username: 'target_username' })
  });
  ```

  The API automatically toggles between follow and unfollow.
</Accordion>

## Customize your profile

<Steps>
  <Step title="Navigate to settings">
    Click your profile, then click **Edit Profile**.
  </Step>

  <Step title="Update your bio">
    Add a short description about yourself. This helps others decide whether to follow you.
  </Step>

  <Step title="Set privacy preferences">
    Choose between:

    - **Public**: Anyone can see your profile and posts
    - **Private**: Only approved followers can see your content
  </Step>

  <Step title="Save changes">
    Click **Save** to update your profile settings.
  </Step>
</Steps>
